# 五子棋游戏 (Gomoku)

一个简单而美观的五子棋网页游戏，使用纯 HTML/CSS/JavaScript 实现。

## 游戏规则

- 黑棋先行，双方轮流落子
- 先在横、竖、斜任意方向连成五子的一方获胜
- 棋盘大小为 15x15
- 支持平局判定（棋盘填满且无人获胜）

## 使用方法

### 在线运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。

### 本地运行

```bash
# 克隆项目
git clone https://github.com/你的用户名/gomoku.git
cd gomoku

# 在浏览器中打开 index.html
```

或使用本地服务器：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需先安装 http-server)
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 游戏操作

- 点击棋盘格子放置棋子
- 点击"重新开始"按钮重置游戏

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计（Grid 布局、渐变效果、动画）
- **JavaScript** - 游戏逻辑（ES6 模块语法）

## 核心功能

### 游戏逻辑
- 棋盘初始化与渲染
- 棋子放置与移动历史记录
- 四个方向的胜负判定（水平、垂直、主对角线、副对角线）
- 获胜棋子高亮显示

### 状态管理
- 当前玩家跟踪
- 游戏结束判定
- 状态信息实时更新

## 特性

- 响应式设计，适配桌面端和移动端
- 获胜棋子高亮动画
- 游戏状态实时显示
- 美观的渐变背景和棋子效果
- 移动历史记录功能
